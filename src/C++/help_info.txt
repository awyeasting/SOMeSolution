Usage: somesolution.exe [-h] [-N Training iterations] [-o OUT] [-i INPUT]
			[-s Square Neurons] [-d Display] [width] [height] [source]

Generates a SOM from a given file source or loads a pretrained SOM from a file. 
Can also display SOMs using a variety of display methods.

Positional Arguments:
	width	The width of the SOM.
	height	The height of the SOM.
	source	THe source data for the SOM to be trained on.

Optional Arguments:
	-h, --help			Show this help message and exit
	-N, --trainingiterations	TRAINTING ITERATIONS. The number of iterations 
					to do in training.
	-i, --input			INPUT. The input file for the SOM's node weights
					(Does not retrain loaded map).
	-s, --squareneurons		SQUARENEURONS. Whether the neurons in the map should
					square. Otherwise it is hexagonal. NOT IMPLEMENTED
	-d, --display			DISPLAY. The display method to be used on the SOM. 
					Current usable display methods: topology, input-planes.
					If no argument after flag, then defaults to topology. NOT IMPLEMENTED
